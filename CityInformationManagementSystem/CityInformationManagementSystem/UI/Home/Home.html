Country Entry

In Country Entry section one can save his country name and country detail information. User can upload video, picture, link and also can change the font style. User can see his information in a data table which he saved before. You can go to Country Entry section from here.

City Entry

In City Entry section one can save city name and city details information like no. of city dweller, cities weather, location according country. User can upload video, picture, link in about field and also can change the font style. User can see his information in a data table which he saved before. You can go to City Entry section from here.

View Cities

In View Cities section one can see his city list according city name or country name. He will see all the information which he saved before. You can go to View Cities section from here.

View Countries

In View Countries section one can see his country list according country name. He will see all the information including total no. of dweller in that country. You can go to View Countries section from here.
